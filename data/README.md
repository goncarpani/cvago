# CV parametrizado (JSON)

El generador de CV **solo** usa datos que estén en `profile.json`. No inventa skills, empresas ni fechas.

## Estructura

- **personal**: nombre, email, teléfono, ubicación, links (LinkedIn, GitHub, portfolio).
- **experience**: lista de trabajos con empresa, rol, fechas, descripción breve, logros (bullet points), tecnologías usadas.
- **education**: títulos, institución, año, notas opcionales.
- **skills**: técnicas (nombre, nivel opcional) y blandas (lista).
- **languages**: idioma y nivel.
- **certifications**: nombre, emisor, año.
- **preferences**: modalidad, industrias, etc. (opcional, para filtros).

Todos los campos de texto que aparezcan en el CV generado deben salir de aquí.
